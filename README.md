Mittach!
========

Installation
------------

### Mac-User

    brew install md5sha1sum

### Anforderungen

* Python >= 2.5
* [pip](http://www.pip-installer.org/en/latest/installing.html)
* virtualenv `sudo pip install virtualenv`

### Setup

Umgebung anlegen:

    make init
    make secret

Server Starten:

    source activate
    make server

[http://localhost:5000/](http://localhost:5000)
