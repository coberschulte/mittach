Mittach!
========

Installation
------------

### Mac-User

    brew install md5sha1sum

### Anforderungen

* Phython >= 2.5
* pip (`sudo easy_install pip`)
* virtualenv (`sudo pip install virtualenv`)

### Setup

Umgebung anlegen:

    make init

Server Starten:
  
    source activate
    make server

[http://localhost:5000/](http://localhost:5000)
